# NewsClean

## Getting Started
To get the project running, you will need an API key from [News API](newsapi.org).

1. **Obtain an API Key:** Register at newsapi.org to get your free developer key.

2. **Setup `local.properties`:** Open the local.properties file in your project root and add the following line:

```

API_KEY="YOUR_ACTUAL_API_KEY_HERE"

```

3. **Build & Run:** Sync Gradle and run the app on an emulator or physical device (API 24+).
